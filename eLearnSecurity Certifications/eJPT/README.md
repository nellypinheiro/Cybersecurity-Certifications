# eJPT – eLearnSecurity Junior Penetration Tester

The **eLearnSecurity Junior Penetration Tester (eJPT)** certification validates foundational penetration testing skills across networking, system exploitation, web application security, and post-exploitation techniques.

This repository documents the hands-on labs and Skill Check CTFs completed as part of the eJPT learning path.

---

## Skills Covered
- Network reconnaissance and enumeration
- Linux and Windows exploitation
- Web application penetration testing
- Credential attacks and hash cracking
- SMB and service misconfiguration exploitation
- Privilege escalation
- Post-exploitation techniques
