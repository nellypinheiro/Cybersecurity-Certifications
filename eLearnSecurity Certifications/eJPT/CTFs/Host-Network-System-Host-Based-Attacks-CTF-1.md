## Overview
This Skill Check Lab focused on system and host-based attack techniques targeting Windows environments. The objective was to identify weaknesses in host configurations, authentication mechanisms, and exposed services that could be leveraged to gain unauthorized access or escalate privileges.

The lab required applying a structured attack methodology against multiple hosts without guided solutions.

---

## Skills Practiced
- Host-based attack techniques
- Authentication and credential attack concepts
- Windows system enumeration fundamentals
- Identification of misconfigurations in host environments
- Multi-target analysis and prioritization

---

## Tools Used
- Nmap
- Hydra
- Cadaver
- Metasploit Framework

---

## Key Learning Outcomes
- Improved understanding of host-based attack vectors
- Hands-on experience attacking Windows systems in a controlled environment
- Reinforced importance of weak credential detection
- Better insight into lateral movement and privilege escalation fundamentals
