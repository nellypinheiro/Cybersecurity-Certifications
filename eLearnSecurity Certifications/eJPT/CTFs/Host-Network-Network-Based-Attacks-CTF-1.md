## Overview
This Skill Check Lab focused on network-based attack analysis and post-exploitation investigation through network forensics. The objective was to analyze captured network traffic to identify indicators of compromise, malicious activity, and infected hosts within a compromised environment.

Using a provided packet capture file, the lab emphasized identifying attacker behavior, compromised systems, and forensic artifacts relevant to incident response.

---

## Skills Practiced
- Network traffic analysis
- Post-exploitation investigation
- Network forensics techniques
- Identification of Indicators of Compromise (IOCs)
- HTTP, DNS, NetBIOS, and PowerShell traffic analysis

---

## Tools Used
- Wireshark

---

## Key Learning Outcomes
- Improved ability to analyze malicious network traffic
- Practical experience investigating compromised hosts
- Understanding how malware communicates over the network
- Identification of infected users, systems, and artifacts via packet analysis
