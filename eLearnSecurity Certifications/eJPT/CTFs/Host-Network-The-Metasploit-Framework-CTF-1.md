## Overview
This Skill Check Lab focused on exploiting Windows-based systems using the Metasploit Framework. The objective was to identify misconfigurations, weak privilege management, and exposed services in a Windows environment, then leverage them to retrieve hidden flags.

The lab emphasized practical use of Metasploit modules combined with manual investigation techniques to navigate Windows directories and escalate access.

---

## Skills Practiced
- Windows service enumeration
- Exploitation using Metasploit Framework
- MSSQL service interaction
- Privilege escalation awareness
- Manual investigation of Windows file system

---

## Tools Used
- Nmap
- Metasploit Framework
- MSSQL client tools

---

## Key Learning Outcomes
- Hands-on experience exploiting Windows services
- Improved understanding of Metasploit workflows
- Identification of sensitive Windows directories
- Awareness of risks related to misconfigured accounts and privileges
