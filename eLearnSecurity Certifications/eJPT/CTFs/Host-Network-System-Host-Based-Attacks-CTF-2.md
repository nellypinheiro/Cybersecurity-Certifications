**Platform:** INE / eLearnSecurity

**Focus Area:** System & Host-Based Attacks (Linux)  

---

## Overview
This Skill Check Lab focused on system and host-based attack techniques targeting Linux environments. The objective was to identify weaknesses in system configuration, exposed services, and access controls that could be leveraged to gain unauthorized access or escalate privileges.

The lab required applying host-based attack methodology across multiple Linux targets without guided solutions.

---

## Skills Practiced
- Linux host-based attack techniques
- System and file system enumeration
- Identification of misconfigurations in Linux environments
- Authentication and access control weaknesses
- Privilege escalation fundamentals

---

## Tools Used
- Nmap
- Burp Suite
- Metasploit Framework

---

## Key Learning Outcomes
- Improved understanding of Linux host-based attack vectors
- Hands-on experience enumerating and attacking Linux systems
- Reinforced importance of file system and permission analysis
- Better insight into privilege escalation paths in Linux environments
