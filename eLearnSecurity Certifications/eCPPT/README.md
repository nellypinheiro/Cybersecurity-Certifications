# eCPPT – eLearnSecurity Certified Professional Penetration Tester

The **eLearnSecurity Certified Professional Penetration Tester (eCPPT)** certification validates advanced, real‑world penetration testing skills across network, system, web application, and client‑side attack scenarios.

This repository documents the hands‑on labs and graded Skill Check Capture the Flag (CTF) challenges completed as part of the eCPPT learning path.

---

## Skills Covered
- Network reconnaissance and enumeration
- Linux and Windows exploitation
- Web application penetration testing
- Client-side and phishing-based attacks
- Credential harvesting and hash cracking
- Service misconfiguration exploitation
- PowerShell for offensive security operations
- Post-exploitation fundamentals
