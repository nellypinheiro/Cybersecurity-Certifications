# ICCA Skills Summary

- Cloud Basics — Understanding fundamental cloud computing principles.
- Cloud Deployment Models — Exploring different ways cloud services can be deployed.
- Cloud Management — Learning how cloud environments are maintained and optimized.
- Cloud Shared Responsibility Model — Defining security roles between cloud providers and users.
- Cloud Services & Providers — Examining the major cloud platforms and their offerings.
- Cloud Identity & Access Management (IAM) — Managing access and security within cloud environments.
